# tyuiu_parse

Небольшой проект для **парсинга расписания** с сайта ТИУ (страница расписания группы) и **отображения** результата в виде HTML-таблицы.

## Что делает

- **Парсер** (`api/api.js`) по таймеру скачивает страницу расписания, декодирует `cp-1251`, извлекает таблицу и сохраняет данные в `parsedData/data.json`.
- **Витрина** (`index.html` + `index.js`) читает `parsedData/data.json` и рендерит таблицу в браузере.

## Стек

- Node.js (ES Modules)
- `axios` — загрузка страницы
- `cheerio` — разбор HTML
- `iconv-lite` — декодирование `cp-1251`
- `browserify`/`watchify` + `babelify` — сборка фронта в `dist/bundle.js`

## Установка

```bash
npm install
```

## Запуск парсера (обновление `parsedData/data.json`)

Парсер работает как долгоживущий процесс и **раз в час** обновляет файл `parsedData/data.json`.

```bash
node api/api.js
```

Если нужно изменить группу/год/параметры расписания — правьте URL в `api/api.js` (параметры запроса в `config.url`).

## Сборка и просмотр таблицы в браузере

Сборка:

```bash
npm run build
```

Режим наблюдения (пересборка при изменениях):

```bash
npm run watch
```

Просмотр:

- Откройте файл `index.html` в браузере (он подключает `dist/bundle.js`).
- Таблица строится из данных в `parsedData/data.json`.

## Структура проекта

- `api/api.js` — парсер расписания, запись JSON
- `parsedData/data.json` — результат парсинга
- `index.js` — генерация HTML-таблицы из JSON
- `index.html` — страница для вывода таблицы
- `dist/bundle.js` — собранный бандл (вывод `browserify`)
